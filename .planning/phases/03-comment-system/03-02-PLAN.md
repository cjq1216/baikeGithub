    2. **在 `delete_lemma` 路由后追加** `delete_comment` 路由(基于 `delete_lemma` L28-38 的同款 query/flash/delete/commit/redirect 模式):
       ```python
       @admin.route('/comment/<int:comment_id>/delete', methods=['POST'])
       @admin_required
       def delete_comment(comment_id):
           comment = Comment.query.get(comment_id)
           if comment is None:
               flash('删除失败！评论不存在')
               return redirect(request.referrer or url_for('apple.home'))
           db.session.delete(comment)
           db.session.commit()
           flash('删除成功！')
           return redirect(request.referrer or url_for('apple.home'))
       ```

    关键点:
    - `@admin_required` 装饰器已定义(L11-25,Phase 2 D-20);非 admin 调用 → `abort(403)` 走统一错误页

    - 模板侧跳转用 `url_for('admin.delete_comment', comment_id=comment.id)`
