<objective>
Migrate the codebase from Python 2.7 to Python 3.11+ by removing all Python 2 idioms (`reload(sys)`, `sys.setdefaultencoding`) and replacing the `mysql-python` (MySQLdb) dependency with `mysqlclient` in `requirements.txt`. Rebuild the virtualenv from scratch and verify the app boots cleanly.

Purpose: Phase 1's first half (INFRA-01, INFRA-02). The runtime must work on a modern Python interpreter before any bug-fix work in Plan 1.2 can be validated. The two halves of Phase 1 are orthogonal: this plan changes the interpreter and dependencies, Plan 1.2 changes the application code.

Output:
- A `requirements.txt` listing only Python 3-compatible packages.
- `app/__init__.py` and `app/api/__init__.py` free of `import sys; reload(sys); sys.setdefaultencoding('utf8')` blocks.
- A working `.venv` (or equivalent) on Python 3.11+ with all packages installed.
- A smoke verification confirming imports and boot succeed.
</objective>
